# Biblioteca-Virtual
O propósito deste código é gerenciar e organizar os livros e clientes de uma 
biblioteca. Nesse contexto, é possível realizar o cadastro dos livros com 
titulo, autor, código e número de páginas, bem como realizar o cadastro
dos usuários desta biblioteca com dados protegidos através do encapsulamento,
um dos fundamentos da Programação Orientada a Objeto (POO). Além disso, o sistema permite 
organizar o empréstimo de livros a partir da relação entre o número do cartão de acesso e o nome 
do usuário, possibilitando identificar quem realizou cada empréstimo. Por fim, também é possível 
registrar a devolução dos livros e consultar todos os livros e clientes cadastrados no sistema.
